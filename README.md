### Overview
This Power BI dashboard analyzes Netflix data to provide insights into viewing trends, content popularity, and user preferences.

### Key Features
- **Total Viewership KPI:** Displays the total number of views across all content.
- **Content Type Distribution (Pie Chart):** Shows the distribution of movies vs. TV shows.
- **Top Genres (Bar Chart):** Visualizes the most popular genres based on viewing hours.
- **Content Rating Distribution (Donut Chart):** Displays the distribution of content ratings.
- **User Ratings Analysis (Line Chart):** Tracks changes in user ratings over time.
- **Top 10 Countries by Viewership (Map):** Geographical analysis of viewership.

### Getting Started
To view the Netflix Dashboard:
1. Download and install Power BI Desktop from [Microsoft's official website](https://powerbi.microsoft.com/desktop/).
2. Clone or download the Power BI file (`Netflix_Dashboard.pbix`) from this repository.

### Usage
1. Open Power BI Desktop.
2. Open the downloaded `Netflix_Dashboard.pbix` file.
3. Interact with the dashboard:
   - Explore viewership trends by genre, content type, or geographical region.
   - Analyze user ratings and content popularity.
   - Gain insights into Netflix content performance and audience preferences.
